Nuxeo Test Case Management Marianne Addon
=========================================

This plugin provide a service to manage specific period of time.

Currently the service is only able to return a date given a specific label and a date.

In the future this plugin will be able to
- register a type of document that needs to be dealt with within a certain period of time
- register listener when the period is over
- possibly create task for this documents with task's deadline set
- register a specific period for  each document.
- the period take into account the off days
- contribute a way to know the off days
